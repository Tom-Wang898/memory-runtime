# Memory Stack Public Export Bundle

Generated from sanitized staging exports.

This bundle can contain:

- `codex-memory-overlays`
- `memory-palace-project-tools`

## Recommended use

- review exported manifests first
- run an absolute-path sweep before publishing
- transplant selected files into the current public repo by default
- create separate mirror repos only if independent ownership is truly needed

## Why this exists

It gives you a safe review buffer between:

- private local source checkouts
- public GitHub history

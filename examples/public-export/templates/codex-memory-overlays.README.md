# codex-memory-overlays

Sanitized project-memory overlays exported from a private Codex rules checkout.

## Contents

- project memory bootstrap and context scripts
- recording and optimization helpers
- durable-memory prompt notes
- Codex workflow overlay material

## Important placeholders

- `${CODEX_REPO_ROOT}`
- `${MEMORY_PALACE_ROOT}`
- `${HOME}`

Resolve them only after cloning into your own environment.

## Publish checklist

1. Review `PUBLIC_EXPORT_MANIFEST.json`.
2. Verify no private absolute paths remain.
3. Add repository metadata and license if you publish this as a standalone mirror.
